# Audits

One folder per audit, kebab-case named after the subject (e.g. `get-right/`).
Each holds an `audit.md` plus any supporting assets (screenshots, exported data).

These are documentation only — they live outside `src/pages/` and `public/`, so
Astro never builds or serves them. Commit progress as you go; git history is the
review trail.
